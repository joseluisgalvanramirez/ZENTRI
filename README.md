# Zentri - Plataforma Educativa con IA

Zentri es una plataforma web educativa impulsada por Inteligencia Artificial, diseñada para democratizar el acceso a la educación. Surge como solución para jóvenes y adultos que desean estudiar de forma autodidacta, pero enfrentan barreras económicas, laborales o físicas.

## 🎯 Objetivo

Mejorar el rendimiento académico y ampliar las oportunidades educativas y profesionales mediante rutas de estudio personalizadas y orientación vocacional generadas por IA.

## 🚀 Características

- **Evaluación Inteligente**: Análisis del perfil del usuario mediante IA
- **Rutas Personalizadas**: Generación automática de planes de estudio
- **Dashboard Interactivo**: Seguimiento de progreso en tiempo real
- **Módulos de Aprendizaje**: Contenido estructurado por áreas
- **Sistema de Progreso**: Guardado automático del avance

## 📋 Requisitos

- Node.js (v14 o superior)
- npm (v6 o superior)
- SQLite3
